My Network Project - A Customized Router Setup

What is PFSense?

pfSense is a free and open source operating system for routers and firewalls.

pfSense can be installed on most commodity hardware, including old computers and embedded systems. pfSense is typically configured and operated though a user-friendly web interface, making administration easy even for users with limited networking knowledge. Generally, one never needs to use terminal or edit config files to configure the router. Even software updates can be run from the web UI.

Source: https://teklager.se/en/pfsense-introduction-open-source-router-firewall/

My Reasons For Building out Router

The main reason for this project is to learn about the network side of things. As a system's administrator, this has really helped me understand the network side of IT. PFSense has a powerful firewall that can be customized to block traffic from leaving or entering your network. Another feature I use is DNS and DHCP. I send all my dns request to PFSense and issue out IPs for the network. I use something called pfBlocker that can block ads for all devices on your network. pfBlocker doesn't block all ads since it can only sniff traffic that's not encryped. PFSense supports VLANs and VPNs. Additional modules can be added from the libary.

This is My Network Setup. I have a Micro Computer on the Bottom running the PFSense OS. The Router is connected to a unmanaged switch and divided from there. I also have an Unifi Access Point connected to the switch.
