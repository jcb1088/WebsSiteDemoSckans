Setting Up OneDrive

Why Do Backups?

As a system’s administrator, one of the most important considerations is to backup all your data and configurations. The loss of data is devastating and results in long hours to rebuild the service and leaves the possibility of angry users calling who depend on the data. Backups from the user perspective is just as important but setting up a backup solution isn’t a concern until after data is lost. When asked by friends and family if it was possible to grab some important file their system that’s completely broke, I always ask if the user did a backup but receive the same answer, No. This is my solution to how I backup my files to the cloud and created a roaming profile that synchronizes my data on all my systems. Note that it's also important to keep a local backup as ransomware attacks could make it impossible to access your data.

Why OneDrive?

OneDrive is the system that I use. You can use Dropbox, Google Drive, ICloud or other user storage solutions.

OneDrive Requirements

Requirements
An active Microsoft Account
Microsoft OneDrive Install on the Operating System
Network Connection
For Windows: 32-bit or 64-bit version of Windows 10, Windows 8/8.1, or Windows 7
For Mac: OS X 10.12 or later

Confirm OneDrive is Running

Before configuring OneDrive, make sure that Microsoft OneDrive is running. For Window's based systems, the OneDrive icon is displayed in the system tray when running.

Open OneDrive Settings

On the Window's system's tray, right click on the OneDrive's icon and select the settings option.
